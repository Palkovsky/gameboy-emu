# Gameboy Emulator

[![](http://img.youtube.com/vi/zxdKIvfu8_0/0.jpg)](http://www.youtube.com/watch?v=zxdKIvfu8_0)

# Features

What works:

* CPU
* Graphics: background, window, sprites
* DMA
* MBC1, MBC2, MBC3 mappers
* Timer
* Controls
* APU

What doesn't work:

* Link cable
* SGB(Super Game Boy)
* CGB(Gameboy Color)
